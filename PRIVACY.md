# 럭키픽 개인정보처리방침 초안

럭키픽은 로또 번호 생성을 위한 앱이며, 사용자의 개인정보를 직접 수집하거나 외부 서버로 전송하지 않습니다.

## 수집하는 정보

- 앱은 이름, 연락처, 위치, 기기 식별자 등 개인을 식별할 수 있는 정보를 수집하지 않습니다.
- 저장한 번호 조합과 최근 생성 기록은 사용자의 기기 내부 저장소에만 저장됩니다.

## 정보 이용

- 저장한 번호 조합은 보관함과 통계 표시 기능을 제공하기 위해서만 사용됩니다.
- 해당 정보는 앱 삭제 또는 보관함 삭제 시 제거될 수 있습니다.

## 제3자 제공

- 럭키픽은 사용자 정보를 외부 업체나 제3자에게 제공하지 않습니다.

## 책임 고지

- 앱에서 생성되는 번호는 무작위 추천이며 로또 당첨을 보장하지 않습니다.
- 사용자는 번호 선택과 구매에 대한 책임을 직접 부담합니다.

## 문의
