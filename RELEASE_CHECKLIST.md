# 럭키픽 구글플레이 출시 체크리스트

## 앱 패키징

- Android Studio 프로젝트 또는 Trusted Web Activity 래퍼 생성
- 패키지명 결정: 예시 `com.yourname.luckypick`
- 앱 이름: `럭키픽 - 로또 번호 생성기`
- 앱 아이콘: `icon.svg` 기반으로 512px PNG, adaptive icon 제작
- 버전 코드와 버전명 설정

## 스토어 등록 자료

- 짧은 설명: `고정 번호와 제외 번호로 나만의 로또 조합을 만들어 보세요.`
- 전체 설명: 생성, 보관함, 최근 통계, 오프라인 사용 가능, 개인정보 미수집 강조
- 휴대전화 스크린샷 2장 이상
- 512px 고해상도 앱 아이콘
- 피처 그래픽 1024x500
- 개인정보처리방침 URL

## 정책 및 고지

- 당첨 보장 표현 금지
- 무작위 추천 및 오락/편의 도구임을 명확히 고지
- 앱 안에서 복권 구매, 결제, 배당, 당첨금 관리, 구매 대행, 실제 베팅 보조 기능을 제공하지 않기
- 실제 로또 구매 기능을 넣는 경우 관련 법률 및 스토어 정책 별도 검토
- 광고 또는 분석 SDK를 추가하면 개인정보처리방침과 데이터 보안 양식 갱신
- 개인정보처리방침은 공개 URL로 제공하고, 앱 안에서도 접근 가능하게 구성

## 출시 전 QA

- 생성, 저장, 삭제, 복사 기능 테스트
- 모바일 화면 360px, 390px, 430px 폭에서 버튼과 텍스트 겹침 확인
- 오프라인 실행 확인
- 앱 삭제 후 로컬 저장소 초기화 동작 확인
